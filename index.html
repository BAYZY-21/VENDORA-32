الثيمات
🛠️ خريطة الأدوات التقنية
🐦
Flutter (Dart)
واجهة التطبيق — Frontend
إطار العمل الرسمي من Google لبناء تطبيقات Android/iOS احترافية. واجهة سريعة، سلسة، وجاهزة لـ Google Play بنقرة واحدة.
flutter_bloc (State) firebase_core cloud_firestore firebase_auth firebase_storage shared_preferences google_maps_flutter dio (HTTP) cached_network_image
🐍
Python + FastAPI
الخادم الخلفي — Backend
FastAPI هو أسرع إطار Python لبناء APIs. سيُستخدم لمنطق العمل المعقد، الإحصائيات، الإشعارات، وإدارة التوكنات بأمان.
FastAPI SQLAlchemy Pydantic JWT Auth Firebase Admin SDK Pillow (ضغط صور) FCM (إشعارات) PostgreSQL
🔥
Firebase
قاعدة البيانات والتخزين
Firebase يوفر Firestore للبيانات الفورية، Storage للصور، و Authentication لتسجيل البائعين والموظفين فقط — مما يقلل التكلفة.
Firestore (Real-time) Firebase Storage Firebase Auth FCM (Push) Firebase Hosting
🚀
Google Play & Deploy
النشر والرفع
Flutter ينتج ملف APK/AAB جاهز مباشرةً للرفع. Python Backend يُرفع على Render أو Railway مجاناً في البداية.
flutter build appbundle Play Console Render.com (Backend) Docker GitHub Actions (CI/CD)
